# Foto60 - Modern React E-commerce Store

A production-ready, minimalist poster e-commerce application built with React, Zustand, and modern web technologies.

## 🚀 Features

- **Modern React Architecture** - Functional components with hooks
- **Global State Management** - Zustand for cart, favorites, and user data
- **Shopping Cart System** - Full cart management with persistence
- **Search & Filter** - Real-time search with debouncing
- **Dark Mode Support** - Theme switching with localStorage
- **Product Reviews** - In-app review system with ratings
- **Poster Builder** - Custom poster creation
- **Responsive Design** - Mobile-first approach
- **Analytics Ready** - Google Analytics integration
- **Error Boundaries** - Graceful error handling
